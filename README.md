# portfolio
https://ksmai.github.io/portfolio/

## Start dev server
```
git clone https://github.com/ksmai/portfolio
cd portfolio
yarn
yarn start
```

## Build for production
```
yarn run build
```

## Run tests
```
yarn test
```
